# constants

All constants required for coding.

## Installation

```bash
npm install @arun-me/constants
```

## Usage

```
import {aplphabet,aplphabetSmall,aplphabetCaps} from "@arun-me/constants"


# map aplphabet array

 {alphabet.map((letter) => (
          <p>{letter}</p>
        ))}

# output
    a
    b
    c
    .
    .
    .
    z
```
